# CodeAlpha URL Shortener

## Description
A URL Shortener backend application built using Node.js, Express.js, MongoDB, and Mongoose.

## Features
- Shorten long URLs
- Redirect short URLs to original URLs
- MongoDB database integration
- REST API

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose

## API Endpoints

POST /shorten
Creates a short URL.

GET /:shortCode
Redirects to the original URL.
